# E-Commerce-CQRS-Outbox-Pattern

Projeto pessoal de estudo com o objetivo de aplicar na prática os padrões **CQRS** (Command Query Responsibility Segregation) e **Outbox Pattern** em uma arquitetura de microsserviços.

> Projeto em construção.

---

## Stack

- **Java 21**
- **Quarkus 3.x**
- **PostgreSQL**
- **RabbitMQ**
- **Flyway**
- **JWT (RSA 2048)**
- **Docker + Docker Compose**
